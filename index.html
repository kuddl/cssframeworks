Definition

Grids sind eine unsichtbare Grundlage um visuellen Zusammenhalt in einem magazin oder Webseite sicherzustellen. ätzlich unterteilen sie den zur verfügungstehenden Platz in horizontale und verticale gleichbleibende Einheiten, welche Text, Bilder, Überschriften usw. angeordnet werden könnnen.

Vorteile

  • Konsistentes Design
  • Reduktion von CSS-Fehlern
  • Funktioniert wie ein Wireframe
  • Relativ unkompliziert Änderungen durchführen
  • Cross-Browser-Support
  • Schnelle Entwicklung von Prototypen

Gemeinsamkeiten / Unterschiede der Systeme

  • gut Teilbare Spaltenzahl (12,16,24)
  • Horizontale Raster, Vertikale "Freiheit"
  • Zeilenbasiert: die Zeile braucht (meist) ein "clear"
  • Kommen ohne JavaScript aus
  • erste und letzter "Baustein" macht meist "Probleme"
  • Px, Em, Prozent Grids
  • Pre-Prozessoren bieten neue Möglichkeiten (Semantic Grids)
